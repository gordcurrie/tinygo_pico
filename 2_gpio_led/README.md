# LED with button

# Equipment

* Raspberry Pi Pico
* 3 * male to male jumper wires
* 1 * LED
* 1 * 330 ohm resistor (orange, orange, brown, gold)

# Assembly

* gnd pin -> blue power rail
* blue power rail -> LED 1
* LED 2 -> resistor 1
* resistor 2 -> GP14
